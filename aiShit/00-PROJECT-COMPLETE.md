## 🎯 Ready to Deploy

### Choose Your Method:

**⚡ Docker (Easiest - 3 minutes)**
```bash
scp -r . user@server:/opt/roon-dashboard/
ssh user@server && cd /opt/roon-dashboard && docker-compose up -d
# Access: http://your-server-ip/
```

**⚙️ Traditional Linux (5 minutes)**
```bash
scp -r . user@server:/opt/roon-dashboard/
npm install && cd client && npm run build && cd ..
sudo systemctl start roon-dashboard
# Access: http://your-server-ip/
```

## 💡 Pro Tips for Deployment

1. **Use Docker** - Easiest option, no dependency management
2. **Set correct Roon Core IP** - Update .env if not 100.90.5.35
3. **Monitor logs** - `sudo journalctl -u roon-dashboard -f`
4. **Approve in Roon** - Settings → Extensions → Approve
5. **Play music** - Wait 30+ seconds for data to appear
